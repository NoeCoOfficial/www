This is a SvelteKit website using shadcn-svelte (powered by [`sv`](https://github.com/sveltejs/cli).)

## Developing

1. Install

```bash
pnpm install
```

2. Start dev server

```bash
pnpm dev
```

## Building

To create a production version:

```bash
pnpm build
```
